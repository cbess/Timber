Timber
======

iOS and Mac app logging made easy.

## Macros

CBTLog(Level, Tag, Message, ...)  
CBTDebugCode(BLOCK)  
CBTDebugMark()

#### Log macros

CBTLogVerbose(MSG, ...)  
CBTLogDebug(MSG, ...)  
CBTLogInfo(MSG, ...)  
CBTLogWarn(MSG, ...)  
CBTLogError(MSG, ...)  
CBTLogDebugError(`NSError`)  

#### Log macros w/ tagging

CBTLogVerboset(TAG, MSG, ...)  
CBTLogDebugt(TAG, MSG, ...)  
CBTLogInfot(TAG, MSG, ...)  
CBTLogWarnt(TAG, MSG, ...)  
CBTLogErrort(TAG, MSG, ...)  

## Notes

See `CBTimber.h` for more details.
